## How to use this as a reference

These files show what AgentReady produces on a well-structured minimal Python project. Use them to:

- Understand the format and content of each generated artifact
- Calibrate expectations before running on your own repository
- Check that the skills and hooks sections are correctly populated for a pytest + pip + CI project
- See a real eval report with pass/fail breakdown by category
